# IVN Reference Integrations

These examples show the two primary adoption paths for IVN:

- `cursor-mcp.json` wires IVN into an MCP-capable editor
- `http-ingest.mjs` sends knowledge into a local `ivn serve --http` process

These files are included in the published npm package so maintainers can inspect or copy them after `npm install ivn`.

Use them as starter snippets for real project templates, internal tooling, or onboarding docs.
